# 1. Introduction<a name="introduction"></a>
<p>
  Morph OS is a project that aims to create a program that "morphs" any type of file to any other type of file. 
  
  Right now the project only has started so it only has a few types of conversions but i plan to increase those as time goes on, I'm a physics student so i do not have a lot of time on weekdays so i will be working on this on weekends.
</p>

# 2. Features<a name="features"></a>

- Already implemented :
  - Mass modification of files 
  - PDF → PNG
  - PNG → PDF
  - OGG → MP3
  - Explanation of every feature

- Plan to implement:
  - Music (lossy)
    - MP3 → OOG , WMA , AAC (m4a/apple)
    - WMA , AAC → MP3 
  - Music (lossless)
    - FLAC ↔ AIFF , WAV , ALAC (m4a/apple)
  - Music (Lossy ↔ Lossless)
    - FLAC ↔ MP3
  - Film
    - MP4 ↔ AVI , MOV , WMV , MKV , FLV , WEBM , MPEG
  - Image
    - PNG ↔ JPEG / JPG , GIF , BMP , TIFF / TIF , WebP , SVG , HEIF / HEIC

    (Have already confirmed all of them are possible)

- Want to implement :
  - Document
    - DOCX ↔ PDF , ODT , TXT
    - TXT ↔ MD , RTF , ODT
  - Spreadsheets
    - XLSX / XML ↔ CSV , ODS
    - CSV ↔ JSON
  - Presentations
    - PPTX ↔ PDF , ODP
  - Scientific / technical documents
    - LaTeX → PDF
    - BibTeX → RIS , XML
  - Structured data
    - JSON ↔ XML , YAML , CSV
    
    (Not sure if all of them are possible)
